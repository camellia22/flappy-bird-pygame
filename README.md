# Flappy Bird (Pygame)
A simple Flappy Bird clone built using Python and Pygame.

## Features
- Animated bird
- Random pipe generation
- Collision detection
- Score tracking
- Start and replay buttons
- Sound effects

## Installation
Clone the repository:
git clone https://github.com/camellia22/flappy-bird-pygame.git
Install dependencies:
pip install -r requirements.txt
Run the game:
python main.py
